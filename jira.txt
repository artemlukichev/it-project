Интеграция системы управления проектами Jira

В текущей версии образовательной среды отсутствует интеграция с системой управления проектами. Это ограничивает возможности командной работы и отслеживания задач.

Требования:

Добавить возможность интеграции с Jira для создания, обновления и отслеживания задач.
Реализовать аутентификацию пользователей через API Jira.
Обновить документацию по использованию интеграции.
